Bulk Email Draft Generator (Excel + Outlook)

Create up to 10 Outlook email drafts at a time using Excel. Emails open as drafts — nothing is sent automatically.

What it does

  • Up to 10 drafts per run
  • Unique recipient + subject + optional attachment per row
  • Uses Outlook desktop to create drafts

Requirements

  • Microsoft Excel (desktop)
  • Microsoft Outlook (desktop)
  • Windows
  • Macros enabled
Download free tool Need a custom version?

When opening: click Enable Editing, then Enable Content (macros). Always review drafts and attachments before sending.
