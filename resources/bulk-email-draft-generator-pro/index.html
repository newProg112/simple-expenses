Bulk Email Draft Generator (Pro)

Create up to 50 Outlook email drafts per run using Excel. Emails open as drafts — nothing is sent automatically.

Pro version · 50 drafts/run

What it does

  • Up to 50 drafts per run (ideal for batch processing)
  • Unique recipient + subject + optional attachment per row
  • Uses Outlook desktop to create drafts
  • Graceful error message if Outlook slows or runs out of memory

Why a 50-draft limit?

Opening large numbers of Outlook draft windows can be limited by system memory. Running in batches helps keep Outlook responsive and reliable.

Requirements

  • Microsoft Excel (desktop)
  • Microsoft Outlook (desktop)
  • Windows
  • Macros enabled
Buy & download (£9) Need help?

When opening: click Enable Editing, then Enable Content (macros). Always review drafts and attachments before sending.
